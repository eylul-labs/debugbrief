# Approval Policy

This project is designed to be operated by Eylul with Serdar as owner.

## Eylul Can Do Without Additional Approval

- Write code locally
- Create documentation
- Research competitors and keywords
- Draft landing pages, posts, emails, and support replies
- Create local mockups and demo scripts
- Prepare release notes and changelogs
- Organize tasks in Workboard

## Approval Required

- Creating external accounts
- Publishing to VS Code Marketplace, GitHub, Product Hunt, Reddit, X, LinkedIn,
  or any public site
- Sending messages to users, customers, communities, or companies
- Spending money
- Connecting payment accounts
- Making legal, tax, or company identity claims
- Uploading private code or private data to third-party services

## Operating Principle

Prepare aggressively, publish deliberately.
