# VS Code Marketplace Draft

## Name

DebugBrief

## Tagline

Turn terminal errors into AI-ready debugging briefs.

## Short Description

Generate structured debugging prompts from errors, stack traces, build failures,
and related source files directly inside VS Code.

## Long Description

DebugBrief helps developers give better context to AI coding tools.

Select an error log, stack trace, or copied file path, then run DebugBrief from
the command palette. It creates a clean Markdown brief with the failure, likely
reproduction command, detected language/test signals, suggested first checks,
and related source files when they can be resolved locally.

Use the generated brief with Codex, Cursor, Claude Code, ChatGPT, or a bug
report.

## Commands

- `DebugBrief: Create Debug Brief From Selection`
- `DebugBrief: Create Debug Brief From Clipboard`
